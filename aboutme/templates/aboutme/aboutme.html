Projects

Capybara Investments Platform


The Capybara Investments project marked a significant advancement in my Django skills, emphasizing the importance of creating visually appealing and accessible applications. It blended my existing expertise with a budding interest in JavaScript, enhancing my web development skills. This project delved into custom strategy optimization, integrating real-time data and advanced analytics for tailored solutions. Leveraging yfinance for real-time stock updates and Chart.js for interactive data visualization, the project offered an engaging analysis experience. Django's robust backend was crucial for tracking transactions and analyzing user activity. The implementation of AJAX for asynchronous operations provided a seamless, reload-free user experience, demonstrating modern web development efficiencies.

Report Generator


As the lead developer, I designed and implemented an advanced custom report generation system. This project expanded my skills in Django and underscored the importance of creating visually appealing and accessible applications. It also sparked my interest in JavaScript, which I am currently learning to further enhance my web development capabilities.

Laboratory Management System


A comprehensive laboratory management system developed for the Polish Academy of Sciences. This system integrates various modules such as laboratory schedule management, reagent tracking, sample management, and scientific protocol generation, thereby enhancing the operational efficiency of neurobiology labs.

Interactive Online Auction System


Explore the thrill of online bidding with our Interactive Online Auction System. Built on Django, this platform offers users a seamless experience, from browsing auctions to adding favorites to their watchlist. Registered members can bid, comment, and even manage their auctions with a user-friendly interface. The admin page provides comprehensive control over listings, facilitating an organized and dynamic auction environment.

FastFood Snake


This Snake game was one of my early Python projects, created as a fun way to learn the basics of programming. It offers a classic gaming experience with a Python twist. I decided to share it because of its simplicity and nostalgic charm. The game uses Turtle for graphics and Pygame for sound, providing an immersive experience with various levels, unique backgrounds, and sound effects. It's a lighthearted and entertaining introduction to Python for beginners.
